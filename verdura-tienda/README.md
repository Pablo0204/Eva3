# 🥬 Verdura Tienda - Sistema de Venta de Verduras

Sistema completo de e-commerce para venta de verduras desarrollado con **Angular 17 + Node.js Express + MySQL**.

## 🚀 Inicio Rápido

### Terminal 1 - Backend (MySQL + API)
```bash
npm run backend
```

### Terminal 2 - Frontend (Angular)
```bash
npm start
```

Accede a: **http://localhost:4200**

## 🛠️ Stack Tecnológico

- **Frontend**: Angular 17 + TypeScript 5.4
- **Backend**: Express.js 5.2
- **Base de Datos**: MySQL 8.0+
- **Gestión de Estado**: RxJS + BehaviorSubject
